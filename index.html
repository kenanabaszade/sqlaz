Haqqımızda

İT sahəsində 20 illik təcrübə: Biz, İT sektorunda 20 illik təcrübəmizlə, müxtəlif sənaye sahələrində lider layihələrə imza atmışıq.

Texnika üzrə PhD elmi dərəcə: Dərin tədqiqat və texniki biliklərimiz PhD dərəcəsi ilə təsdiqlənmişdir.

Oracle üzrə Beynəlxalq sertifikatlar: Tanınmış Oracle sertifikatlarımız, platforma üzrə geniş təcrübəmizi göstərir.

Oracle Certified Master-ın ən yüksək dərəcəsi: Bu unikal sertifikat, mürəkkəb problemlərin həllində mükəmməlliyimizi nümayiş etdirir.

Xarici və yerli biznes tərəfdaşlarımız: Geniş biznes şəbəkəmiz bizə milli və beynəlxalq səviyyədə iş imkanları yaradır.

Uğurlu layihələr: Keçmiş uğurlu layihələrimiz sayəsində, müştəri məmnuniyyəti və keyfiyyətli xidmətlərimizi təsdiqləmişik.

Müəllimlərimiz

Səmədov Samir Bəxtiyar oğlu

2005-ci ildə Bakı Dövlət Universitetinin Tətbiqi Riyaziyyat və Kibernetika fakültəsinin İnformatika ixtisasını fərqlənmə diplomu ilə bitirib. 2008-ci ildə magistratura pilləsini fərqlənmə diplomu ilə tamamlayaraq aspiranturaya daxil olmuş və texnika üzrə fəlsəfə doktoru dərəcəsini almışdır. 15 ildən artıq fərqli İT sahələrində (Bank, Card & Data Processing, dövlət sektoru) çalışmışdır. 2013-cü ildən Oracle şirkətinin elit qrupunun üzvü olaraq Oracle 11g Certified Master (OCM) titulunu qazanmışdır.

Oracle imtahanları üzrə uğurlar:

  • 120-007: Oracle9i: SQL

  • 1Z0-031: Oracle9i: DBA Fundamentals I

  • 1Z0-033: Oracle9i: Performance Tuning

  • 120-101: PL/SQL proqramlaşdırma

  • 1Z0-032: Oracle9i: DBA Fundamentals 2

  • 1Z0-040: Oracle Database 10g: Yeni xüsusiyyətlər

  • 1Z0-047: Oracle Database: SQL Certified Expert

  • 1Z0-050: Oracle Database 11g: Yeni xüsusiyyətlər

  • 1Z0-046: Oracle Database 10g: Linux-da Oracle İdarəçiliyi

  • Oracle 11g Certified Master 2 Practice Days Exam

  • 1Z0-060: Oracle Database 12c-ə keçid

  • 120-931: Oracle Autonomous Database Cloud 2019 Certified Specialist

  • 1Z0-1072: Oracle Cloud Infrastructure 2019 Architect Associate

  • 1Z0-1085: Oracle Cloud Infrastructure Foundations 2020

Həmçinin SQL və digər texnologiyalar üzrə geniş təcrübəyə malikdir. Ətraflı məlumat üçün: www.ocm.az

Səmədov Ramin Bəxtiyar oğlu

Texnika üzrə Fəlsəfə Doktoru, 15 ildən artıq İT sahəsi üzrə çalışır.

Bakı Dövlət Universitetinin Tətbiqi Riyaziyyat və Kibernetika fakültəsinin tələbəsi olub. Bakalavr, maqistratura və doktorantura pilləsini uğurlu şəkildə məhz həmin Universitetdə başa vurub.

Azərbaycanın ən böyük İT sahələrində (Telecom, Bank və Processing), müxtəlif rəhbər vəzifələrdə çalışıb. Ən iri Bank sistemləri sayılan Oracle Flexcube və Processing sistemi Tranzaxis TX miqrasiyalarında yaxından iştirak edib, uğurlu nəticələr əldə edib.

Oracle verilənlər bazasının inzibatcılığını, PL/SQL proqramlaşdırma dilini və SQL sorğu dilini mükəmməl bildiyinə görə - Oracle verilənlər bazası üzrə ekspertdi. Bundan əlavə JAVA üzrə proqramlaşdırma ilə məşğuldur. Həmçinin Linux ƏS və Cloud texnologiyaları üzrə dərin biliklərə malikdir.

Hal-hazırda elmi-praktiki və innovativ fəaliyyətini davam etdirir. Bir neçə elmi əsərlərin müəllifidir və ölkəmizi bir necə xarici konfranslarda uğurla təmsil edib.
